# Duckies

A web application for exploring Akizuki-class destroyers.

## Features
- Scrollytelling experience
- Multi-language support (ID/EN)
- Detailed ship data

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
